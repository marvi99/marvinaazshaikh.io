About Me ;

From the moment, I wrote my first line of code and printed a hello world output in the console I knew I was hooked on.
Computer Science for me was hacking into social media accounts, and that’s what I had in mind while accepting my offer to this program.
Unfortunately, it wasn’t at all what I imagined. Going into my first semester’s first class Cp104 I realized how blindsided I was about CS.
Luckily with the help of my friends and lots of youtubers I successfully download IDLE and Eclipse and started coding. Gradually, I started to enjoy the thrill of finding the missing comma in my code due to which my code wouldn’t compile. I started to enjoy the adrenaline rush I would get when I would see the exact output I hoped for in the console.

Today I continue to venture out and explore different technologies, learn new coding techniques, and experiment with several programming languages to further improve my software skills.
